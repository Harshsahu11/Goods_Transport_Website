🚚 Goods Transport Service Platform

A web-based platform designed to simplify the process of booking and managing goods transportation services.
This system connects customers who need to transport goods with transport providers, enabling efficient logistics management.

The platform helps users book transport services, track shipments, and manage delivery details through an easy-to-use interface.

🌟 Features
🚚 Transport Booking

Users can book transport services for goods delivery.

Select pickup and destination locations.

Choose vehicle type based on goods size.

📦 Shipment Management

Manage shipment details easily.

Track booking information and delivery status.

👤 User Accounts

Secure user authentication.

Users can manage their booking history.

📍 Order Tracking

Track goods delivery status.

View transport details in real-time.

📱 Responsive Design

Fully responsive UI.

Works across desktop, tablet, and mobile devices.
